![FitPro Preview](./preview.png)

## Características

- **Diseño responsive** - Mobile-first, funciona en todos los dispositivos
- **Animaciones fluidas** - Scroll animations y transiciones CSS
- **Modo oscuro/claro** - UI adaptable con gradientes modernos
- **Validación de formularios** - Login modal con validación en tiempo real
- **Toggle de precios** - Switch mensual/anual con animación
- **Navegación suave** - Smooth scroll entre secciones

## Stack Tecnológico

| Tecnología | Versión | Propósito |
|------------|---------|-----------|
| HTML5 | Semántico | Estructura |
| CSS3 | Moderno | Estilos y animaciones |
| JavaScript | Vanilla | Interactividad |
| Google Fonts | Inter | Tipografía |

## Quick Start

```bash
# Clona el repositorio
git clone <repo-url>

# Navega al directorio
cd fitpro-landing

# Abre el archivo en tu navegador
open index.html
```

No requiere dependencias ni build tools.

## Estructura

```
fitpro-landing/
├── index.html      # Página principal
├── css/
│   └── styles.css  # Estilos y design system
└── js/
    └── app.js      # Interactividad y animaciones
```

## Secciones

1. **Hero** - CTA principal con estadísticas
2. **Features** - 5 características con IA
3. **Cómo funciona** - 3 pasos
4. **Precios** - 3 planes (Básico, Pro, Elite)
5. **Testimonios** - Prueba social
6. **CTA Final** - Conversión
