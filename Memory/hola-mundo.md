# Hola Mundo

Este archivo fue creado para probar el MCP de GitHub.

## Estado

- MCP GitHub configurado correctamente
- Esperando token de GitHub en variable GITHUB_TOKEN

## Operaciones disponibles

El servidor MCP de GitHub soporta:

- **Archivos:** Crear, actualizar, eliminar archivos
- **Repositorios:** Crear, bifurcar (fork)
- **Issues:** Crear, actualizar, listar, añadir comentarios
- **Pull Requests:** Crear, fusionar, revisar
- **Búsquedas:** Código, issues, repositorios, usuarios
- **Branches:** Gestión de ramas
- **Commits:** Historial de commits
